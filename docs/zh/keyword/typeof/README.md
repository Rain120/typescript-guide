## 定义 <Badge text='Typescript typeof 关键字' />

:::theorem typeof
用于获取变量的声明类型。
:::

**Note:**

- 参数:  `typeof` 只接受我们正在检查其变量类型的变量的名称, 该变量的名称在右侧, 而左侧则没有。

- 返回: `类型 (Type)` 的值。

## 使用

```ts
function getType(val: any) {
  return typeof val;
}

getType('name'); // string
```
